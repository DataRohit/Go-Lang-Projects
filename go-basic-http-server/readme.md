# Go Basic HTTP Server

## Packages Used

1. net/http
2. log

## Routes

-   `/` - Serving the base `index.html` file
-   `/form.html` - Serving the `form.html` file
-   `/hello` - Writing a hello message to the DOM
-   `/form` - Handling the submit POST requests from the `/form.html`
